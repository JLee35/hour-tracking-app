Welcome to the Scout Lake Construction Payroll Application

Use this application to log hours for yourself and your employees.

The process of this application is as follows:

  • Enter employee, job site, and pay period information
  • Enter general hourly information for each day (time in/out, lunch start/stop)
  • Enter specific hourly information for each day (individual code phases and time spent)
  • Review, sign, and submit timecard
This application will walk you through these steps. Select Get started to continue.
